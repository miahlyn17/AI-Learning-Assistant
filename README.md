# AI-Learning-Assistant
AI Learning Assistant is an intelligent platform that helps community to learn and explore topics efficiently. It uses four specialized agents — Professor, Advisor, Librarian, and Assistant to provide explanations, guidance, resources, and quick support. The system is powered by Flask for the backend, Tailwind CSS for a modern interface, and GROQ for fast data retrieval.

# Features
- Professor: Explains topics step-by-step with examples.
- Advisor: Suggests study plans and learning strategies.
- Librarian: Provides curated resources like books, tutorials, and articles.
- Assistant: Offers quick answers, summaries, and supports other agents.
- Responsive UI: Modern interface built with Tailwind CSS.
- GROQ: Efficient content retrieval for personalized learning.

# Technologies
- Python
- Flask
- Tailwind CSS
- Groq
